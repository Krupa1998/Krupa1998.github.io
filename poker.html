PokerTrax App


FEATURES

  1. It is simple to use and has sophisticated UI.
  2. It is an app made for mobile.
  3. App that can be used by all age group

ABOUT

This is a poker tracking application. It stores history of all the poker games played by user and keeps track of their performance and displays it in graph format for easy usability. It has blogs section for users to learn more about poker and a glossary section for basic skills to learn before playing poker.
